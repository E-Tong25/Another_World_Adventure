# Another_World_Adventure

Still in progress! - as of March 3,2023

This is my text-based, isekai, adventure game project for the 2022 Spring session of Code Louisville, within the Software Development Track

In this game, you will be able to create a character and be reborn into a new world full of monsters and magic. You will be able to join a guild, fight enemies with your choice weapon, heal with potions, and explore different areas.

I hope you enjoy!
